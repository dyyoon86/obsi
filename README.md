# ObsidianVault

Codex/OMX/Hermes 장기 작업을 위한 Obsidian 기반 외부 기억장치입니다.

## 주요 폴더

- `000 Projects/` — 프로젝트별 brief, plan, tasks, decisions, progress-log, memory
- `010 Profile and Memory/` — 사용자 프로필과 장기 메모리 요약
- `020 Hermes Skills/` — Hermes 로컬 스킬 export
- `030 Codex OMX/` — Codex + OMX + Obsidian MCP 사용 가이드

## 운영 원칙

작업 시작 전 관련 프로젝트 폴더를 읽고, 작업 완료 후 tasks/progress-log/decisions/memory를 갱신합니다.

# 사용자 프로필 / 작업 기준

생성: 2026-05-29 09:12:01

## 핵심 정체성
- 사용자는 한국어로 주로 소통한다.
- 한국 AI YouTube 채널 `투두TV` / `@todo_chn`을 운영한다.
- 선호 방향은 추상적인 AI 전망보다, 직접 만든 AI 실무 자동화 스킬/파일/워크플로우 무료 배포다.
- 훅은 결과/궁금증/약속을 앞에 두고, caveat는 뒤로 보내는 구성을 선호한다.

## 콘텐츠 운영 방향
- 한 소스를 Shorts / Threads / 커뮤니티 / Notion DB로 다중 활용하는 워크플로우에 관심이 있다.
- ionflow 계정은 concise casual Korean Threads-style AI news posts를 선호한다.
- 기본 구조: 뉴스 핵심 한 줄 → “이게 무슨 말이냐면” → 쉬운 설명 → 변화/의미 → 개인 인사이트/짧은 caveat.
- 긴 카드뉴스보다 보조 이미지 1장을 선호한다.

## SaaS/사업 검증 방향
- validation-first 사고를 선호한다.
- 툴을 먼저 팔기보다 직접 운영해 proof/data를 만든 뒤 도구화하는 접근을 선호한다.
- 쇼핑쇼츠 툴은 소싱/카피 리스크 때문에 현재 보류 상태다.
- 야구 직관 SaaS `직캘` / `직관캘린더` 아이디어를 검토 중이다.

## 응답/작업 선호
- 한국어, 실무형, 바로 실행 가능한 결과물을 선호한다.
- 거창한 설명보다 실제 파일/템플릿/스크립트/검증 결과를 선호한다.
- Codex/OMX/Obsidian 패턴은 장기 작업의 외부 기억장치로 쓰는 방향이 적합하다.

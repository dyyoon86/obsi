# Evan Carmichael fit check for 투두TV

Session context: user asked whether Evan Carmichael / @evancarmichael should count as a 골채널 after a benchmark analysis.

## Decision

Do **not** treat Evan Carmichael as a primary 골채널 for 투두TV. Treat it as a **format/reference channel**.

## Why it is not a primary gold channel

투두TV’s current positioning:

- Directly built AI 실무 자동화 skills/files/workflows
- Claude Skills / HWPX / reports / meeting notes / YouTube analysis / Notion content DB
- Free practical templates and workflow packages
- Viewer value: “I can use this at work today”

Evan Carmichael’s positioning:

- Self-improvement / motivation / success rules
- Famous-person and authority-based content curation
- #BelieveNation daily motivation habit
- Viewer value: belief, emotional energy, aspirational success principles

The viewer job-to-be-done differs:

- Evan: “Help me believe I can succeed.”
- 투두TV: “Give me a concrete AI workflow/file/skill I can use.”

## What to copy

Use Evan only for reusable packaging patterns:

- `Top 10 Rules` → `AI 실무 자동화 10가지 규칙`
- `ALL OF X Explained in N Minutes` → `Claude Skills/HWPX/Notion 자동화 완전정리`
- `No BS, No Fluff` → `개념 말고 바로 파일/워크플로우로 보여드립니다`
- Authority/front-loaded proof → use credible work problem, before/after, demo result, or source channel data rather than celebrity clips
- Daily habit/community framing → “하루 하나의 실무 자동화 스킬” rather than “믿음 충전”

## What not to copy

- Broad motivational topics
- Famous-person clip curation as the core value
- “성공한 사람들의 습관” style generic self-help
- Authority-only videos without a practical downloadable/demo output

## Example 투두TV transformations

- `Claude Skills의 거의 모든 것 — 실무 자동화용으로만 정리했습니다`
- `AI 회의록 자동화, 초보자가 가장 많이 망하는 7가지 이유`
- `HWPX 문서 자동화의 거의 모든 것 — No 개념설명, 바로 실전`
- `유튜브 채널분석 자동화, 제가 직접 쓰는 워크플로우 공개`
- `AI 콘텐츠 DB 만드는 법 — Notion 하나로 소재→쇼츠→Threads까지`

## Future analysis instruction

When a large channel is not niche-aligned, explicitly classify it as:

1. Primary 골채널
2. Adjacent 참고채널
3. Format-only reference
4. Avoid/copy-risk channel

Then explain what to copy and what not to copy. The user values this fit check more than a generic channel summary.

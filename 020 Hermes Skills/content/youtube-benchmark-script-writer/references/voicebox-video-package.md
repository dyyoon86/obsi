# VoiceBox / AI voice-cloning video package reference

Session-specific reference for turning the 에딭초이 VoiceBox + HyperFrames source video into a 투두TV-style production package.

## Strategic angle

Do not frame as pure tool 소개 or copy the original `5분만에 내 목소리 복제` angle. 투두TV version should be:

> 보이스박스 퀄리티는 생각보다 좋다. 다만 긴 대본을 한 번에 넣으면 발음/호흡/리듬이 흔들릴 수 있으므로, 실전에서는 짧게 쪼개서 생성하는 워크플로우가 필요하다.

## Core script beats

1. **Cold open with AI-generated narration**: `이 나레이션은 제가 직접 녹음한 게 아닙니다...`
2. **Face-cam reveal**: user explains the voice was generated from their own sample.
3. **Problem**: recording is a bottleneck even when scripts are AI-assisted.
4. **Workflow**: voice sample → VoiceBox → AI narration → split long scripts → optional video draft.
5. **Claude Code demo**: install/setup through Claude Code; ask it to write README, commands, error fixes.
6. **Short sentence test**: quality is surprisingly good.
7. **Long script test**: long input can fail or produce awkward rhythm; do not overclaim.
8. **Practical workflow**: split into 20–40 second chunks, regenerate only bad chunks, stitch in editor.
9. **Optional HyperFrames extension**: call it 영상 초안 자동화, not final automation.
10. **Final evaluation**: short narration useful; long narration needs chunking and human review.
11. **CTA**: share VoiceBox checklist, Claude Code prompt, chunking template, ethics checklist.

## Ethics note

Always include:

- Use only your own voice or explicitly permitted voices.
- Do not clone others for impersonation or misleading commercial use.
- Treat AI voice as a production aid, not a deception tool.

## Directing style

Because the user plans to shoot with face included, output should include:

- face-cam moments: hook, why tested, quality evaluation, long-script issue, final verdict, CTA
- screen-recording moments: folder setup, Claude Code prompt, VoiceBox install, sample upload, short/long generation, chunked script files
- audio examples: original voice, short AI voice, long awkward segment, usable final segment

## Key phrasing

- `짧은 문장 기준으로는 생각보다 제 목소리 느낌이 꽤 났습니다.`
- `문제는 긴 대본을 넣었을 때였습니다.`
- `긴 대본 통째로 X, 짧게 나눠서 생성 O.`
- `AI에게 한 번에 다 맡기는 게 아니라, 작업 단위를 잘게 쪼개서 디렉팅하는 겁니다.`
